# basic_scraper
Basic web scraper

Overview
- This project is a simple web scraper designed to extract information from websites. It demonstrates the basic principles of web scraping, including sending HTTP requests, parsing HTML content, and extracting specific data. This project is built using Python and popular libraries like requests and BeautifulSoup.

Features
- Send HTTP requests to web pages.
- Parse HTML content using BeautifulSoup.
- Extract specific data elements (e.g., titles, links, images).
- Save the extracted data to a CSV file.

Prerequisites - Before you begin, ensure you have met the following requirements:
- Python 3.6 or higher installed on your machine.
- Pip package manager for Python.
